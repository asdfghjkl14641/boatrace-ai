## 最終判定

### パターン別結論

| パターン | 判定 | 根拠 |
|---|---|---|
| I: モデル全体楽観 | **部分的に真** | 全体 ratio=1.0 だが高 p 側で 5-9% 楽観 |
| II: PL 構造破綻 | **棄却** | 1 着→2 着→3 着 の精度が妥当 |
| III: 月別変動不安定 | **現象としては真 (構造問題ではない)** | 型分布・hit1% は安定、変動はオッズ起因 |
| IV: 閾値調整で黒字化 | **棄却** | 全閾値で max ROI=0.87、1.0 到達不能 |

### 根本原因

**複合要因**:
1. **ブックマーカー控除率 (vig) ≈ 25%** (Z ≈ 1.33 ⇒ 1/Z ≈ 0.75)
2. **高 p 側で 5-9% のモデル楽観** — EV 計算上の edge を実質的に消費
3. **120 通り全買い目の中で EV>1.1 が 55k 件** — 選択圧が弱く、本当の edge を抽出できていない
4. **月別ランダム性が大** — 短期は ±20pt 変動が正常範囲

### 改善方向 (俺の判断用材料)

1. **キャリブレーション適用** (Isotonic/Platt on high-p bins)
   - 高 p 楽観を補正 → EV 計算の精度向上
   - 期待効果: +3-5pt 程度 (理論上限)
2. **閾値見直し**
   - 最適値は EV=1.20-1.50 (ROI 0.86-0.87)
   - **単独では 1.0 未到達** → キャリブレーションとの組み合わせが前提
3. **型3 単独戦略**
   - **非推奨**: CI が 1.0 を含み、月別変動極端
4. **その他**
   - 2 連単・2 連複への切替 (vig が低い可能性)
   - Stop-loss / 損失最小化戦略 (黒字を諦め、赤字幅を管理)
   - より強い edge 指標 (例: 下位実績ベット以外を除外)

### 推奨アクション

**最優先**: **キャリブレーション (Isotonic Regression) を高 p 領域に適用**し、
**閾値 EV=1.30 に設定**した上で再バックテスト。
それでも ROI<1.0 なら、**モデル改善よりもゲーム理論的な限界** (vig が強すぎる) として
3 連単は見送り、他券種に展開することを検討。
