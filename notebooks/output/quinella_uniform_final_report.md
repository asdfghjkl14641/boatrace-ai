# 2連複 最終運用仕様 (均等配分版)

## 最終運用仕様

```
券種: 2連複 (quinella)
モデル: v4_ext_fixed
キャリブレーション: Isotonic (quinella train fit)
判定: EV ≥ 1.50 AND Edge ≥ 0.02
型フィルタ: no_t3 (型1 + 型2, 型3 除外)
オッズ上限: 10.0
予算: 3000円/レース
配分: 均等 (候補数で割り 100 円単位切り捨て、残は EV 上位補填)
```

## ベット率定義の確定

「7.77%」と「16.53%」の矛盾は分母の違い:

| 分母 | N | ベットレース | ベット率 |
|---|---|---|---|
| (a) race_cards 全レース | 44,064 | 3,419 | 7.76% |
| (b) results JOIN 可能 | 42,456 | 3,419 | 8.05% |
| **(c) odds 有 (実用定義)** | **41,213** | **3,419** | **8.30%** |
| (d) 候補ありレース | 20,679 | 3,419 | 16.53% |

→ **実用的には (c) 8.30%** で統一。モデルが評価できるレース基準。
→ (d) 16.53% は「候補を持つレースのうち戦略が買うレース率」で意味違い。

## 買い目数分布 (重要発見)

| n点買い | レース数 | 割合 |
|---|---|---|
| **1点買い** | **3,258** | **95.29%** |
| 2点買い | 158 | 4.62% |
| 3点買い | 3 | 0.09% |

→ **ほぼ全部 1 点買い!** だから配分方式 (EV比例/ハーフケリー/均等) で差が出なかった。
→ 実装は **均等 (最もシンプル)** で十分。
