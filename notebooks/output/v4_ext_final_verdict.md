## 最終判定: **パターン β (構造的限界確定)**

### 根拠
- **過学習は解消**: Phase C の test ROI 1.14 は偶然。真の性能は 0.94-0.95。
- **サンプル増加効果**: train 22k → 290k (13倍)、test 16k → 42k (2.6倍)
- **全戦略で ROI < 1.0**: train でも最高 0.984、test でも最高 0.949
- **vig 25% の壁**: v4 モデルの edge が控除率を超えない

### 定量:
- Phase C の過学習減少: 最大 **-94%** (戦略 4 で +0.15 → +0.01)
- ROI 上限: train **0.98**, test **0.95**
- 運用可能 (ROI ≥ 1.0 AND CI 下限 ≥ 0.95) な戦略: **0 / 6**

## 3連単での結論

**3連単は v4 / v4_ext_fixed モデルで構造的に黒字化不可**

以下のいずれかしか道はない:
1. **他券種 (2連単・2連複・3連複)** — vig が 3連単より低い可能性
2. **モデル根本再設計 (v5)** — 新特徴量 (天候直前、気配変化、展示タイム差分) で edge 拡大
3. **ノンリニア化** — GBM/Neural で表現力向上
4. **3連単撤退** — 構想メモ段階の戦略に戻る or 運用諦める
